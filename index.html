Front End Projects

Front End Capstone

TheBibleApp

I created this project to help engage children between the ages of 3 - 6. I struggle to get my children to listen to the bible being read to them, and this is the solution I came up with. I use simple animations(using ng-animate) to hold the childs attention while the verses are read to the child. This project combines my knowledge of CSS, HTML, Javascript, Angular 1, and Firebase.

Local Weather

Local Weather

I created this project to fetch weather conditions from the openweather api. I would then allow the user to choose a date and location and I would display the weather conditions for that location. I also used firebase to register a user and login, so the user could come back to the website at any time and see what search they have completed in the past. This project combines my knowledge of CSS, HTML, Javascript, Angular 1, firebase, and I used an API for the first time.

Checkers

Play Checkers

I created this project to push the limits of what I understood about javascript. This project taught me how to add and remove event listeners \ based on what the user had clicked. This project combines my knowledge of CSS, HTML, and Javascript.
